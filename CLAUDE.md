# AI Repository — CLAUDE.md

## Overview
Web applications for physics education hosted at psd1.net (Pasco School District).
Contains AI chat interfaces, student grade pages, physics tools, and email tracking.

## Primary Languages
HTML, PHP, JavaScript (frontend/backend), Python (data sync)

## Deployment
- Static files: GitHub Pages (auto-deployed via `.github/workflows/static.yml`)
- Dynamic PHP: cPanel hosting at psd1.net
- Domain: psd1.net

## Major Projects
| Folder | Purpose |
|--------|---------|
| `claude/`, `claude2/` | Claude AI chat interfaces for students |
| `chatgpt/`, `chatgpt2/` | ChatGPT interfaces |
| `coach2/`, `coach3/`, `coach4/` | Progressive AI coaching tools |
| `gemini/` | Google Gemini interface |
| `email-tracking/` | Email open-rate tracking system |
| `physicsgrades/` | Per-student physics grade pages (158 students) |
| `motionmapmaker/` | Interactive kinematics motion map tool (D3.js) |
| `calculators/` | Projectile motion calculator |
| `birthdays/` | Student birthday database |

## Architecture Pattern
All AI interfaces follow the same pattern:
- `index.html` — student-facing chat UI
- `api-proxy.php` — server-side API proxy (keeps API keys off the client)
- `dashboard.php` (where present) — teacher/admin analytics dashboard
- `student_logs/` (where present) — per-session usage logs

## Student Folders
158 student subfolders named `{firstname}{number}` (e.g., `aaron046/`).
Each contains:
- `index.html` — links to grade and calendar pages
- `{studentID}.html` — grade summary page
- `{studentID}.calendar.html` — assignment calendar

These are auto-generated by `sync_student_pages.py` from the Misc repo.

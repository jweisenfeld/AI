## Purpose
Browser-based physics calculators for student use in class.

## Key Files
| File | Description |
|------|-------------|
| `projectile.html` | Projectile motion calculator (~43KB, fully self-contained) |

## projectile.html
A standalone HTML file with all physics formulas for projectile motion built in.
Students enter known values and the calculator solves for unknowns.
No external dependencies — works offline once loaded.

## Deployment
Static file at `psd1.net/calculators/`.

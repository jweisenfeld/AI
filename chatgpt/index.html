How it works

  1. The browser collects your messages and stores them in memory (and localStorage).
  2. On send, it posts JSON to chat_api.php.
  3. The PHP file loads your API key from /home2/<account>/.secrets/chatgptkey.php.
  4. The response is appended to the chat log and saved for later refreshes.

Open tests.html to run the unit tests.

Cost tip: start with gpt-4.1-mini and ~900 max tokens, then only switch to gpt-4.1 for harder debugging tasks.

Note: model availability depends on your API project and billing tier.

PHP Scripts

  • PHP api-proxy.php — Server-side API proxy (called automatically by the chatbot; you never open this directly)
  • PHP update_models.php — Model auto-healer

    Run this when students start getting errors like "model not available" after Anthropic retires a model. It pings the API to test which Claude versions actually respond, then promotes a working fallback to primary in model_config.json. Each probe costs essentially nothing (1 token max). Open it in the browser with ?key=YOUR_DASHBOARD_PASS appended to the URL — it prints a live log. Under normal circumstances you should never need this; the proxy already falls back automatically.

  • PHP tests.php — Unit test suite (CLI only)

    Run this via SSH after editing api-proxy.php to make sure nothing is broken. Command: php tests.php from the claude/ folder. It is not designed to be opened in a browser — it validates internal logic like model tier checks, request validation, school-hours rules, conversation caps, and config file structure. You don't need to run it unless you've made code changes.

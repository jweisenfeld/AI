Cache Management

  • LOG cache-ping.log — CRON run history (read this first when troubleshooting)
  • LOG gemini_usage.log — Per-request log (check FILE_URI vs NO_FILE column)
  • PHP cache-ping.php — Manual trigger of the daily cron job

    The cron runs this at 3am daily to keep the Pasco Municipal Code alive in the Gemini Files API (files expire after 48 hours). Run it manually only if the health panel above shows the ping is overdue or students are getting NO_FILE errors. Costs a small amount — it re-uploads the full ~3.5MB document. Prints a live log to the browser so you can confirm success.

    Run cache-ping now
  • PHP cache-create.php — Full rebuild (manual setup / disaster recovery)

    Use this only when:
    • Setting up the tool for the first time
    • The cron has been broken for more than 48 hours and the URI has expired
    • You need to switch to an alternate API key (?key=2)
    • You just ran strip-html.php to update the cleaned document

    Same cost as cache-ping. Unlike cache-ping, it also deletes the old file from Gemini before saving the new URI, avoiding stale-file quota accumulation. Verbose output — you'll see "SUCCESS!" when done.

    Run cache-create (recovery) Run with alternate key

Diagnostics

  • PHP list-models.php — List available Gemini models on this API key

    Queries the Gemini API and prints all available models, highlighting which ones support createCachedContent. Use this if you're unsure which model IDs to put in the proxy, or to verify a new key has the expected model access. Append &save=1 to also write a models_cache.json to disk.

  • PHP test-stream.php — End-to-end streaming test

    Sends a short test prompt through the full proxy stack with SSE streaming enabled and prints the response. Use this to confirm the API key is valid, the model responds, and streaming is working before students start a session.

  • PHP download_logs.php — Download all student logs as a ZIP

    Bundles all student_logs/*.txt files into a timestamped ZIP and sends it as a download. Useful for grant reporting, backing up conversations, or offline analysis.

Supporting Files

  • PHP api-proxy.php — Gemini API proxy (called automatically by the chatbot; never open directly)
  • PHP strip-html.php — Converts raw municipal code HTML to token-efficient plain text

    The raw Pasco-Municipal-Code.html is ~9MB / ~3.3M tokens — too large for Gemini's 1M context window. This script strips all HTML tags and outputs a lean Pasco-Municipal-Code-clean.html (~3.5MB / ~875k tokens) that fits. Run it only if you update the source HTML file. After running it, also run cache-create.php to upload the new version to the Files API.

    Run strip-html.php

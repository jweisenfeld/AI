Technical Notes

Inference model
Claude Haiku 4.5 (claude-haiku-4-5-20251001) — fast, cheap RAG synthesis
Embedding model
OpenAI text-embedding-3-small — 1536 dimensions, used at ingest and at query time
Chunk size
~800 tokens per section chunk (single RCW/WAC/USC/CFR section)
Retrieval
Top 8 results via Reciprocal Rank Fusion (k=60) of vector cosine + BM25 full-text lanes
Max output
2,000 tokens per response
Corpus sources
RCW (leg.wa.gov XML), WAC (leg.wa.gov XML), USC (Cornell LII), CFR (eCFR.gov XML)
Vector store
Supabase + pgvector, HNSW index (m=16, ef_construction=64)
Project repo
jweisenfeld/AI — rcw-wac/ (UI + proxy + ingestion)
